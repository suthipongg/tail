Access to the SDK
=====================================

For Full use access to the SDK download the required files from the `Blueprint Lab Website. <https://blueprintlab.com/github-page/>`_.
Use the password provided to you from Blueprint Lab.


.. image:: website_page.png
   :width: 300



1. Open the ZIP File.

2. Copy the ZIP File to the Blueprint_Lab_Software Folder

.. image:: instruction_install_sdk.png
   :width: 600

3. Merge The folders if prompted.

4. Proceed to install the SDK
